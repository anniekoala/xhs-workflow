# Xiaohongshu Publish Checklist

Run this before delivering a final publishing bundle.

## Copy

- Title is specific and not generic.
- Cover text is shorter than the platform title when possible.
- First line enters the topic quickly.
- Caption adds context not already obvious from the media.
- Copy passed `dependencies/humanizer-xhs/SKILL.md`.

## Visual

- Cover is readable in feed size.
- Text does not block important subject matter.
- Visual passed `rules/visual-rules.md`.

## Video

- First 3 seconds have a reason to keep watching.
- Subtitles are burned in if needed.
- BGM is included in the final video by default.
- A no-BGM backup exists when practical.

## Publishing

- Tags cover topic, scene, audience, and long-tail keywords.
- First comment is prepared if useful.
- Interaction question is concrete.
- Final files are in `published/<note>/`.
- Intermediate files stay in `scratch/<note>/`.
