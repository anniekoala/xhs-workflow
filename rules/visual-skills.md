# Visual Skills Registry

Use this file to decide which visual skill/tool to call. Update it when Annie collects better visual skills.

## Current Preferred Tools

| Need | Preferred skill/tool |
|---|---|
| Generate image素材 or background | `imagegen` |
| Design multi-page Xiaohongshu carousel cards | `presentations` or `xiaohongshu-photo-cover` if installed |
| Design cover using real photo + text | `xiaohongshu-photo-cover` if installed, otherwise `presentations` |
| Video title cards / subtitles / motion package | `hyperframes` |
| Audio transcription / subtitles / voice workflow | `hyperframes-media` or `video-use` if installed |
| Timeline-heavy motion | `gsap` |
| Simple browser-native motion | `waapi` or `css-animations` |
| 3D / premium visual scene | `three` or `typegpu` |

## Rule

Do not let the visual skill invent final copy. The source of truth for title, cover text, caption, and tags is `skills/xhs-workflow/SKILL.md`.
